# Arduino-Based Medicine Dispenser

##  Project Overview
This project presents an **automatic medicine dispenser system** designed to help users take medicines on time and reduce human errors.

It uses **RFID technology** for authentication and an **Arduino Uno** as the main controller to manage the system.

---

## Objective
- To provide medicines 24/7 like an ATM system
- To help elderly and patients follow medication schedules
- To reduce human errors in medicine dispensing
- To support rural areas with limited healthcare access

---

##  Components Used
- Arduino UNO  
- RFID Module (RC522)  
- RFID Card  
- LCD Display (16x2)  
- DC Motor  
- Push Buttons  
- Relay Module  
- Breadboard  
- 9V Battery  

---

## Working Principle
1. User scans RFID card  
2. System verifies authorized access  
3. LCD displays instructions  
4. User selects compartment using buttons  
5. Motor opens selected medicine compartment  
6. Medicine is dispensed  

---

##  Key Features
- RFID-based secure access  
- Automated medicine dispensing  
- User-friendly interface  
- Reduces medication errors  
- Helpful for elderly care  

---
